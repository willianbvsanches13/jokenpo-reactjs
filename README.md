## Aplicação mostrando um jogo de Jokenpo

#regras:

- Tesoura corta papel
- Papel cobre pedra
- Pedra esmaga Lagarto
- Lagarto envenena Spock
- Spock dobra tesoura
- Tesoura decapita lagarto
- Lagarto come papel
- Papel desqualifica Spock
- Spock vaporiza pedra
- Pedra esmaga tesoura
